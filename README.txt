1、命令行创建npm的配置文件
E:\code\gulp-demo>npm init

2、添加一个gulp的依赖（实际就是安装gulp包）
E:\code\gulp-demo>npm install gulp --save-dev

3、在项目根目录添加一个gulpfile.js文件，这是gulp的主文件。这个文件名是固定的。

4、在gulpfile中抽象我们需要做的任务


安装的插件包：
npm install gulp-less gulp-concat gulp-uglify gulp-obfuscate gulp-cssnano gulp-htmlmin browser-sync --save-dev
